# Teslo API 
1. Clonar el proyecto
2. Instalar dependencias
```
npm install
```
3. Clonar el archivo .env.template a .env
```
cp .env.template .env
```
4. Levantar la base de datos
```
docker-compose up -d
```
5. Ejecutar el seed
```
http://localhost:3000/api/seed
```
6. Levantar la API
```
npm run start:dev
```
